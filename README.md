# gitclone-cli <shunwang.tan>

### 特性

```
基于node的cli工具，通过命令行获取git仓库代码完成模板项目创建
```

### 使用说明
1. 安装依赖
```bash
$ npm install 
```

2. 将工具注册到全局
```
npm install -g
```

3. 使用工具下载git仓库模板代码生成模板项目
```
git-clone-cli create
git-clone-cli create https://github.com/miccuci/vueBasic.git
```
